# DESIGN

## Purpose

Hold the Knowledge Design System — every standard governing how Alpha Proxima, Axiom Nexus, and OSGMETAPHYSICS knowledge looks, is structured, is colored, and is read.

## Scope

Design standards only. This folder does not contain knowledge content itself (see `../KNOWLEDGE/`) and does not contain governance doctrine (see `../GOVERNANCE/`) — it defines how content and doctrine are *presented*, never what they say.
