# ORGANIZATION

## Purpose

Hold the institutional structure of Alpha Proxima — who owns every responsibility, how Companions relate to Offices, how information and decisions move, and who owns which part of the repository. New folder, created this sprint, sitting alongside `GOVERNANCE/` (which holds the rules) as the place that holds the org chart those rules govern.

## Scope

Structure and mapping only. Does not contain policy (see `GOVERNANCE/`) or content (see `KNOWLEDGE/`).

## Contents

| File | Answers |
|---|---|
| `office-map.md` | What are the fourteen Offices, and what is each one's full jurisdiction? |
| `companion-office-map.md` | Which Companion fronts which Office, and what are the special-role rules for JERANIUM, ORION, VORTEX, ATHENA, SOHMA, GAIA? |
| `information-routing-map.md` | How does knowledge actually move through the repository? |
| `escalation-map.md` | Where does a decision go when it exceeds an Office's own authority? |
