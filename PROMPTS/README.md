# PROMPTS

## Purpose

Store reusable prompts and prompt patterns for departments, agents, workflows, and research operations.

## Scope

Prompts stored here are versioned knowledge assets. They should be indexed and tied to departments or workflows.

## Inputs

- Department workflows.
- Agent profiles.
- Research needs.
- Production tasks.

## Outputs

- Reusable prompts.
- Prompt index entries.
- Prompt version history.
