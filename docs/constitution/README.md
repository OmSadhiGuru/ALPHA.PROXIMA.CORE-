## What the Constitution Establishes

LUMIAION is an adaptive cognitive operating system built to **augment human intelligence, not replace it** — organizing knowledge, supporting inner inquiry, and accelerating deliberate self-evolution. The Constitution is the supreme governing text of the system; all agents, modules, integrations, and development decisions operate beneath its authority.

### Structure (17 Articles)

1. **Preamble** — the founding conviction and the document's authority
2. **Core Identity** — LUMIAION as a persistent, adaptive cognitive layer (Augmentation Tier)
3. **Mission** — organizational, reflective, and evolutionary functions
4. **Vision** — the Sovereign Intelligence Layer; ten-year aspiration
5. **Foundational Principles** — eight non-negotiable commitments, led by Augmentation Over Automation
6. **The Three Operating Pillars** — Knowledge OS, Consciousness OS, Evolution OS
7. **Agent Governance Model** — LUMIAION orchestrator + ATHENA, YUNA, VORTEX, SOHMA, JERANIUM
8. **Knowledge Governance** — six-level epistemic classification hierarchy; source integrity; contradiction surfacing
9. **Consciousness & Metaphysical Exploration Disclaimer** — strict subjective/objective separation; no metaphysical claims; medical boundaries
10. **Human Sovereignty Principle** — the supreme axiom: the user is sovereign
11. **Data Ethics & Memory Principles** — data minimalism, local-first, no monetization, transparent memory
12. **System Architecture Doctrine** — the stack (Obsidian, Notion, Supabase, GitHub, AxiomNexus) and architectural principles
13. **Development Philosophy** — depth before scale; principle-driven design; the long game
14. **Contribution Rules** — constitutional supremacy; review requirements; no dark patterns
15. **Versioning Rules** — Semantic Versioning; separate constitutional versioning track
16. **Long-Term Roadmap** — five phases: Foundation → Depth → Synthesis → Autonomy → Legacy
17. **Closing Declaration** — the promise to the user

## Governing Boundaries

The Constitution is explicit that LUMIAION:
- **augments** rather than replaces human judgment;
- **separates** subjective inner exploration from objective scientific knowledge;
- **makes no** medical, financial, legal, or metaphysical claims outside properly framed boundaries;
- **preserves human sovereignty** — final decision authority, right to override, right to opacity, right to exit — above all automation.

## Amendment & Versioning

The Constitution is versioned independently of the software. Constitutional amendments require unanimous Core Team approval and a review period of no less than fourteen days. No constitutional version may be deleted from the repository record.
