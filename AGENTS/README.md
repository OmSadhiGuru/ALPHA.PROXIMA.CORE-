# AGENTS

## Purpose

Store agent profiles, capabilities, boundaries, and operating instructions.

## Scope

This folder describes agent roles. It does not execute agents or automate decisions.

## Inputs

- Department needs.
- Agent profile templates.
- Capability definitions.
- Boundary rules.

- Capability maps.
