# 11_OPERATIONS

## Purpose

`11_OPERATIONS` is the operational nervous system of the Alpha Proxima Foundation. It describes how offices coordinate, how information flows, how artifacts move through their lifecycle, and how recurring review rhythms preserve institutional continuity.

This area does not create governance authority. It makes institutional workflows explicit, reusable, and easier to operate.

---

## Scope

This office covers operating procedures, review cycles, workflow registries, artifact registries, office coordination, dashboard definitions, metrics, health checks, automation queues, and operational version history.
