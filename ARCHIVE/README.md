# ARCHIVE

## Purpose

Store retired, superseded, or inactive institutional memory.

## Scope

Archived files remain searchable and versioned unless removal is required for security, privacy, or legal reasons.

## Inputs

- Superseded documents.
- Retired project material.
- Deprecated prompts.
- Historical logs.

## Outputs

- Preserved institutional history.
- Supersession references.
- Archive index updates.
