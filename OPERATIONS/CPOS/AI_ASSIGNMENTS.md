## Assignment Rules

- Assign work to departments first, employees second.
- If an employee changes, keep the department and replace only the Current Employee value.
- A department may only claim one active task at a time unless the Founder explicitly approves parallel ownership.
- Before starting, update this file and the source queue file.
- On completion, update the source queue, write a handoff, and pull the next eligible item.
