## The evidence/framework labeling rule (used throughout)

Per the founding constraint, every substantive claim in client-facing material is tagged:

- **[Evidence-based]** — supported by published research in psychology, neuroscience, or behavioral science. Stated with appropriate humility; no overclaiming.
- **[Contemplative / Personal framework]** — drawn from contemplative traditions, metaphysical frameworks, or OSG's own model. Presented as a lens and a practice, **not** as scientific fact.

This distinction is a **trust asset**, not a limitation. Most operators in this space blur the line; OSG's credibility comes from never doing so. The public-facing phrasing of this promise appears in the website copy and the course intro.

## The 30-day priority order (summary — full checklist in file 07)

1. **Week 1** — Lock the flagship offer + price; write/publish sales page; set up checkout + email tool.
2. **Week 2** — Record/assemble course Module 1–2; build welcome + confirmation emails; open the waitlist.
3. **Week 3** — Finish remaining modules or stage them for live delivery; set up community; soft-launch to warm list.
4. **Week 4** — Public launch; run the sales sequence; enroll; onboard first cohort.

**Shipping principle:** You do **not** need all course modules finished to launch. A live cohort lets you build Modules 3–8 *while teaching them*. Launch on the strength of Module 1–2 + the transformation promise.
