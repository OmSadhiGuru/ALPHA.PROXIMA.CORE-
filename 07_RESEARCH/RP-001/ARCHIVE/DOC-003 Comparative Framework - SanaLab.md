## Scope

This document provided a structured comparative framework for seven leading theories of consciousness:

1. Global Neuronal Workspace Theory (GNWT)
2. Integrated Information Theory (IIT)
3. Predictive Processing (PP)
4. Recurrent Processing Theory (RPT)
5. Higher-Order Thought Theory (HOT)
6. Attention Schema Theory (AST)
7. Active Inference (under the Free-Energy Principle)

Each theory was analyzed across nine dimensions: core claim, definition of consciousness, mechanism, brain structures, evidence strength, weaknesses, open questions, AI compatibility, and relevance to Alpha Proxima / LUMIAION.
