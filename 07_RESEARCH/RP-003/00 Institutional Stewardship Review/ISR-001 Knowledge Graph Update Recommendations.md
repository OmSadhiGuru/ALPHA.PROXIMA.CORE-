# ISR-001 — Knowledge Graph Update Recommendations

*This document specifies the knowledge graph additions and cross-program relationship edges identified during the ISR-001 stewardship review. Recommendations are grouped by type: new cross-program edges, missing program-internal edges, and new nodes required. Implementation is assigned to CF-01 (Institutional Architecture) during the RP-003 activation phase.*

## Section 1 — Cross-Program Edges (P1 — Required Before RP-003 Activation)

The following typed relationship edges should be created between RP-001 and RP-002 concept nodes. These edges establish the knowledge graph infrastructure for RP-003.

### Edge Set 1 — Consciousness × Memory Architecture

| Source Node | Relationship | Target Node | Evidence Class | Rationale |
|------------|-------------|-------------|---------------|-----------|
| Consciousness (RP-001) | grounds | Memory (RP-002) | C | Conscious access is the condition under which intentional encoding occurs; memory retrieval requires conscious access |
| Global Neuronal Workspace Theory (RP-001) | instantiates | Working Memory (RP-002) | M | GNWT's global broadcast is the mechanism through which working memory is maintained; the workspace IS the working memory substrate |
| Recurrent Processing Theory (RP-001) | competes_with | Working Memory (RP-002) | M | RPT claims consciousness arises in sensory cortex independently of prefrontal WM; challenges WM-consciousness identity |
| Integrated Information Theory (RP-001) | grounds | LTP - Synaptic Plasticity (RP-002) | E | IIT's Φ is generated by the same synaptic integration that LTP measures; high Φ regions are high-connectivity regions with strong LTP capacity |
| Default Mode Network (RP-001) | supports | Memory (RP-002) | E | DMN is most active during episodic memory retrieval; DMN deactivation correlates with memory encoding failure |

### Edge Set 2 — Predictive Processing as Shared Backbone

| Source Node | Relationship | Target Node | Evidence Class | Rationale |
|------------|-------------|-------------|---------------|-----------|
| Predictive Processing (RP-001) | extends | Engram (RP-002) | E | The engram may encode prediction weights rather than event copies; PP reframes engram as stored prediction, not stored record |
| Predictive Processing (RP-001) | grounds | Pattern Separation and Completion (RP-002) | M | DG pattern separation and CA3 pattern completion are PP mechanisms: DG generates orthogonal representations (distinct predictions); CA3 completes partial cues (prediction from partial evidence) |
| Active Inference (RP-001) | extends | Reconsolidation (RP-002) | E | Reconsolidation is the active inference update: retrieved memory = prior; new context during retrieval = prediction error; restabilization = prior update |

### Edge Set 3 — Consciousness as Key to Reconsolidation

| Source Node | Relationship | Target Node | Evidence Class | Rationale |
|------------|-------------|-------------|---------------|-----------|
| Consciousness (RP-001) | requires | Reconsolidation (RP-002) | E | Reconsolidation is triggered by conscious retrieval; unconscious priming does not open the full reconsolidation window |
| Reconsolidation (RP-002) | requires | Consciousness (RP-001) | E | Reciprocal: the reconsolidation window is a consciousness-dependent event |
| Meta Awareness (RP-001) | grounds | Reconsolidation (RP-002) | S | Meta-awareness (knowing that one is remembering) may be the specific conscious condition that maximizes the reconsolidation update; speculative but mechanistically plausible |
| Attention Schema Theory (RP-001) | extends | Trauma Memory (RP-002) | M | Trauma disrupts the attention schema — the brain's model of its own attentional processes — producing intrusive memories that bypass normal voluntary retrieval routing |

### Edge Set 4 — Contemplative Intersection

| Source Node | Relationship | Target Node | Evidence Class | Rationale |
|------------|-------------|-------------|---------------|-----------|
| 4E Cognition (RP-001) | grounds | Contemplative Memory (RP-002) | M | Enactivism (Varela) is the theoretical framework that gives contemplative practices scientific standing as embodied, embedded, enacted cognitive operations — including memory operations |
| Active Inference (RP-001) | extends | Contemplative Memory (RP-002) | S | Active inference suggests that contemplative practice is a systematic program for reducing precision-weighted prediction error — which at the memory level means reducing the emotional salience-weighting of encoded experiences |
| Consciousness (RP-001) | grounds | Trauma Memory (RP-002) | C | Trauma memory is a disorder of consciousness-memory integration: the memory is encoded during states of extreme arousal that disrupt normal conscious integration, producing re-experiencing rather than recall |

### Edge Set 5 — Hippocampus as Structural Bridge

| Source Node | Relationship | Target Node | Evidence Class | Rationale |
|------------|-------------|-------------|---------------|-----------|
| Global Neuronal Workspace Theory (RP-001) | extends | Hippocampus (RP-002) | E | The hippocampal index theory (Teyler & DiScenna) is a memory-specific implementation of GNWT: the hippocampus binds distributed cortical representations by creating a global index — functionally analogous to GNWT's global broadcast binding of local representations |
| Neural Correlates of Consciousness (RP-001) | grounds | Engram (RP-002) | M | NCCs are the neural basis of conscious experience in the moment; engrams are the persisting record of those neural states; the relationship between a NCC (transient) and an engram (persistent) is a fundamental unresolved question |
| Predictive Processing (RP-001) | grounds | Pattern Separation and Completion (RP-002) | M | Pattern separation (DG) implements prediction diversification; pattern completion (CA3) implements prediction from partial cue — both are PP operations at the cellular level |

## Section 2 — Missing Program-Internal Edges (P2)

### RP-001 Internal Edges Missing

| Source Node | Relationship | Target Node | Evidence Class | Rationale |
|------------|-------------|-------------|---------------|-----------|
| Global Neuronal Workspace Theory | competes_with | Recurrent Processing Theory | M | COGITATE was specifically designed to test these against each other; the competition should be formally graphed |
| Attention Schema Theory | extends | Meta Awareness | M | AST is the theoretical framework; Meta Awareness is the experiential phenomenon it attempts to explain |
| Default Mode Network | contradicts | Global Neuronal Workspace Theory | M | DMN is most active during mind-wandering and self-referential processing — which is precisely when global workspace "ignition" should be minimal; their relationship is complex and not simply supportive |
| Panpsychism | supports | Integrated Information Theory | S | IIT has panpsychist implications (high Φ implies some form of experience in simple systems); this relationship is explicit in Tononi's writing |

### RP-002 Internal Edges Missing

| Source Node | Relationship | Target Node | Evidence Class | Rationale |
|------------|-------------|-------------|---------------|-----------|
| Reconsolidation | extends | LTP - Synaptic Plasticity | C | Reconsolidation is LTP re-engaged at retrieval — the same molecular cascade (NMDA → CaMKII → AMPA insertion) that operates at encoding re-operates at re-encoding; this relationship should be explicit |
| Working Memory | precedes | Engram | C | Only information maintained in working memory has the opportunity for LTP-mediated encoding into long-term memory; working memory is the gateway to engram formation |
| Hippocampus | contains | Pattern Separation and Completion | C | Pattern separation occurs in DG; pattern completion in CA3; both are anatomically within the hippocampal formation — the containment relationship should be explicit |
| Collective Memory | extends | Memory | M | Collective memory is not simply aggregated individual memory (Halbwachs); it is a distinct social construct that operates through shared frameworks — it extends the concept of memory beyond individual neural instantiation |
| Trauma Memory | exemplifies | Reconsolidation | C/P | Trauma memory is the failure-mode exemplar of reconsolidation: traumatic memories that are retrieved are re-encoded with the same emotional intensity rather than being updated — the reconsolidation window fails to normalize the emotional salience |

## Section 4 — Graph Structural Recommendations

### 4.1 Cross-Program Graph Master Index

A cross-program knowledge graph master index should be created at `07_RESEARCH/Cross-Program/Knowledge Graph Master Index.md` linking all RP-specific research graphs and documenting the inter-program typed relationships. Currently, no such document exists — the RP-001 and RP-002 knowledge graphs are fully separate.

### 4.2 Typed Relationship Legend

The following typed relationships are used in this recommendation document and should be standardized in the knowledge graph schema:

| Relationship | Definition |
|-------------|-----------|
| grounds | A → B: A provides the theoretical or empirical foundation that makes B possible or intelligible |
| extends | A → B: A builds upon B, adding scope, depth, or precision without contradicting it |
| instantiates | A → B: A is a specific implementation or instance of the more abstract B |
| competes_with | A ↔ B: A and B make incompatible or partially incompatible claims about the same phenomenon |
| contradicts | A ↔ B: A provides evidence against B; not merely competition but empirical conflict |
| requires | A → B: A cannot occur without B being present or activated |
| supports | A → B: A provides evidence consistent with B without being its theoretical ground |
| exemplifies | A → B: A is a specific case or example of the general principle B |
| precedes | A → B: A must occur before B in causal or temporal sequence |
| contains | A → B: A is the anatomical, conceptual, or structural container of B |

These are the ten typed relationships from the Research Integration Framework. All knowledge graph edges should use this vocabulary.
