# RP-002 — Comparative Framework

> This framework synthesizes and extends the 10-domain comparative matrix from DOC-B (SanaLab). The original DOC-B matrix is preserved in the ARCHIVE; this document is the institutional adaptation for Alpha Proxima use.

---

## Framework Purpose

A single discipline's account of memory is necessarily partial. The comparative framework holds ten accounts simultaneously — not to relativize them, but to identify where they converge (producing high-confidence claims) and where they diverge (revealing genuine open questions or complementary levels of analysis).

---

## The Ten-Domain Matrix

### Domain 1 — Neuroscience

| Axis | Content |
|------|---------|
| **Definition of memory** | Persistent modification of neural circuits through experience-dependent synaptic plasticity |
| **Core models** | LTP/LTD; CLS theory; Standard Consolidation Theory; Multiple Trace Theory; Reconsolidation |
| **Key terminology** | LTP, engram, consolidation, reconsolidation, SWR, theta-gamma coupling, MTL, CA1/CA3/DG |
| **Functional emphasis** | Physical substrate and mechanistic dynamics; circuit-level computation |
| **Primary evidence type** | Electrophysiology, optogenetics, molecular biology, neuroimaging |
| **Key open question** | Engram code; glial coupling; non-synaptic storage; schema integration mechanism |

### Domain 2 — Psychology

| Axis | Content |
|------|---------|
| **Definition of memory** | Information processing system enabling acquisition, storage, and retrieval of experience |
| **Core models** | Multi-store model (Atkinson-Shiffrin); Working memory (Baddeley); Levels of processing (Craik-Lockhart); Tulving's SPI model |
| **Key terminology** | Encoding, storage, retrieval, working memory, episodic, semantic, procedural, implicit, explicit |
| **Functional emphasis** | Behavioral access, modification, and error (false memory, eyewitness) |
| **Primary evidence type** | Behavioral experiments, neuropsychology (lesion studies), reaction time |
| **Key open question** | Reliability of memory; misinformation effect boundaries; relationship between WM capacity and LTM |

### Domain 3 — Education

| Axis | Content |
|------|---------|
| **Definition of memory** | The capacity to retain and apply learned material across contexts and time |
| **Core models** | Spacing effect; Testing effect (retrieval practice); Interleaving; Desirable difficulties (Bjork) |
| **Key terminology** | Encoding, consolidation, retrieval practice, spacing, interleaving, transfer, forgetting curve |
| **Functional emphasis** | Instructional design; maximizing durable learning; minimizing forgetting |
| **Primary evidence type** | Controlled learning experiments; classroom studies |
| **Key open question** | How to scale optimal memory practices to institutional education; individual variability in memory effectiveness |

### Domain 4 — Artificial Intelligence

| Axis | Content |
|------|---------|
| **Definition of memory** | Mechanisms for storing and accessing information across time in computational systems |
| **Core models** | Parametric (weights), Transient (hidden states/context), External (RAG, replay buffers, DNC) |
| **Key terminology** | Parametric memory, context window, experience replay, RAG, catastrophic forgetting, continual learning |
| **Functional emphasis** | Scalable knowledge storage; avoiding catastrophic forgetting; efficient retrieval |
| **Primary evidence type** | Benchmark performance, ablation studies, computational experiments |
| **Key open question** | Can AI systems develop genuine autobiographical memory? How to solve catastrophic forgetting at scale? |

### Domain 5 — Cognitive Science

| Axis | Content |
|------|---------|
| **Definition of memory** | Working memory as real-time information processing gateway to long-term storage |
| **Core models** | Working memory model; Attention gating models; Predictive processing |
| **Key terminology** | Working memory, attentional filtering, cognitive load, executive control, capacity limits |
| **Functional emphasis** | Real-time information processing; attention-memory interface; executive control |
| **Primary evidence type** | Cognitive experiments, neuroimaging, computational modeling |
| **Key open question** | Precise mechanism of theta-gamma coding; relationship between WM capacity and intelligence |

### Domain 6 — Philosophy

| Axis | Content |
|------|---------|
| **Definition of memory** | A form of intentional mental state directed at the past; foundational to personal identity |
| **Core models** | Memory theory of personal identity (Locke); Epistemological theories of memory; Causal theories of memory |
| **Key terminology** | Personal identity, continuity of self, justified belief, veridical vs. non-veridical memory, epistemic warrant |
| **Functional emphasis** | Ontology of memory; epistemic status; relationship between memory and selfhood |
| **Primary evidence type** | Conceptual analysis; thought experiments; interface with empirical results |
| **Key open question** | Can a being without continuous memory have personal identity? What epistemic justification do memories provide? |

### Domain 7 — Anthropology

| Axis | Content |
|------|---------|
| **Definition of memory** | Distributed, socially constituted capacity for cultural transmission across generations |
| **Core models** | Collective memory (Halbwachs); Oral tradition (Ong, Vansina); Sites of memory (Nora); Cultural schema theory |
| **Key terminology** | Collective memory, oral tradition, cultural schema, memorial practice, sites of memory, invented tradition |
| **Functional emphasis** | Intergenerational transmission; cultural continuity; identity construction through shared past |
| **Primary evidence type** | Ethnography, oral history, archival research, comparative cultural analysis |
| **Key open question** | How do digitally mediated collective memories differ from pre-digital forms? How is cultural forgetting structured? |

### Domain 8 — Meditation Research

| Axis | Content |
|------|---------|
| **Definition of memory** | A modifiable relationship to past experience; a dimension of consciousness that can be transformed |
| **Core models** | Attentional deconditioning; Metacognitive recontextualization; Contemplative memory transformation |
| **Key terminology** | Mindfulness, deconditioning, metacognition, non-attachment, present-moment awareness |
| **Functional emphasis** | Transformation of memory's emotional charge; reducing automatic conditioning; therapeutic wellbeing |
| **Primary evidence type** | First-person phenomenology; emerging neuroimaging; behavioral outcome studies |
| **Key open question** | Mechanistic intersection with reconsolidation; whether contemplative effects require specific practice traditions |

### Domain 9 — Trauma Research

| Axis | Content |
|------|---------|
| **Definition of memory** | A disruption in memory's normal reconstructive process; intrusive, involuntary, fragmented recall |
| **Core models** | PTSD model (DSM); Reconsolidation-based therapy; Narrative integration therapy; EMDR |
| **Key terminology** | Intrusion, fragmentation, hyperarousal, reconsolidation window, trauma narrative, EMDR |
| **Functional emphasis** | Therapeutic modification of pathological memory; restoration of narrative integration |
| **Primary evidence type** | Clinical trials; neuroimaging; case studies |
| **Key open question** | Reliable clinical protocols for reconsolidation-based PTSD treatment; long-term durability of therapeutic reconsolidation |

### Domain 10 — Systems Theory

| Axis | Content |
|------|---------|
| **Definition of memory** | Systemic capacity for adaptive change — information retained from past states that modifies future system behavior |
| **Core models** | Autopoiesis (Maturana, Varela); Feedback loops; Emergent memory in complex systems |
| **Key terminology** | Autopoiesis, emergence, feedback, attractor states, path dependency, organizational learning |
| **Functional emphasis** | Systemic resilience and adaptation; organizational learning; institutional memory |
| **Primary evidence type** | Systems modeling; organizational case studies; complex systems analysis |
| **Key open question** | How does collective intelligence in human-AI systems develop memory-like properties? |

## Cross-Domain Convergences

These claims are supported across multiple domains — each converging from different evidence types:

| Convergent Claim | Domains | Evidence Class |
|-----------------|---------|----------------|
| Memory is reconstructive | Neuroscience, Psychology, Anthropology, Trauma | C-001 |
| Context shapes encoding and retrieval | Psychology, Education, Cognitive Science, Neuroscience | C-010 |
| Emotional salience biases storage | Neuroscience, Psychology, Trauma, Contemplative | C-006 |
| Offline consolidation is mandatory | Neuroscience, Psychology, Education | C-005 |
| Memory shapes and constitutes identity | Philosophy, Anthropology, Trauma, Contemplative | Cross-disciplinary [DOC-B] |

## Disciplinary Terminology Bridges

Same phenomenon — different names across disciplines:

| Neuroscience | Psychology | Education | AI |
|-------------|-----------|-----------|-----|
| Reconsolidation | Memory updating | Schema revision | Weight update |
| Hippocampal fast-learning (CLS) | Episodic memory | Novelty encoding | Few-shot learning |
| SWR offline replay | Sleep consolidation | Spaced repetition benefit | Experience replay |
| Adult neurogenesis | Cognitive flexibility | Transfer learning capacity | Fine-tuning / continual learning |
| LTP → LTD (synaptic homeostasis) | Forgetting | Interference & forgetting curve | Catastrophic forgetting |

## Coverage Gaps

Areas not covered by any reviewed source:
- Indigenous/non-Western memory epistemologies (noted in DOC-B as underrepresented)
- Developmental memory (childhood through aging lifespan)
- Comparative animal memory cognition
- Mathematical/computational memory models (beyond AI domain)
- Pharmacological memory interventions
