## Purpose

RP-002 maps human memory across scales — from molecular mechanisms to cultural transmission — to inform the design of LUMIAION's memory architecture and Alpha Proxima's institutional memory practices.

Memory is not peripheral to Alpha Proxima's mission. It is central: LUMIAION's ability to accumulate knowledge, maintain continuity across sessions, and build on prior work depends on a rigorous understanding of how memory functions, fails, and can be designed.

---

## Core Finding

**Memory is reconstructive.** Every act of recall re-creates rather than replays. This is true at the molecular level (reconsolidation), the cognitive level (reconstructive recall), and the cultural level (collective memory as ongoing social construction). The implication for Alpha Proxima: LUMIAION's memory must be designed for active maintenance, not passive storage. Records that are not regularly retrieved and reviewed will drift from accuracy.

## Five Anchoring Insights

1. **Memory is physical.** LTP, synaptic protein synthesis, dendritic spine remodeling, oscillatory dynamics — these are physical processes. Memory design must respect physical constraints (capacity, interference, consolidation timing).

2. **Sleep consolidates.** Systemic consolidation requires the offline processing phase of sleep. For LUMIAION, the Knowledge Writeback Protocol serves this function — but it is voluntary and session-dependent, unlike biological sleep.

3. **Emotion gates encoding.** Emotionally salient events encode more strongly. For institutional memory, this means high-stakes decisions and crises will be remembered; routine operations may not. Explicit documentation is the mitigation.

4. **The past can be revised.** The reconsolidation window means retrieved memories are modifiable. Alpha Proxima's versioned document architecture mirrors this — every retrieved document can be updated; every update creates a new version.

5. **Collective memory is an institution.** Cultural memory is not the sum of individual memories — it is constituted by shared practices, artifacts, and stories. The ALPHA.PROXIMA.CORE- vault IS Alpha Proxima's collective memory. Its integrity is the integrity of the institution's memory.

## Open Frontiers

Four questions at the frontier of memory neuroscience remain unresolved and may have direct implications for AI memory design:

1. **The engram code** — the physical instantiation of a specific memory is unknown
2. **Schema integration mechanism** — how prior knowledge shapes new encoding at circuit level
3. **Non-synaptic storage** — whether epigenetic mechanisms solve the protein turnover paradox
4. **Glial coupling** — the role of astrocytes and microglia in consolidation and pruning
