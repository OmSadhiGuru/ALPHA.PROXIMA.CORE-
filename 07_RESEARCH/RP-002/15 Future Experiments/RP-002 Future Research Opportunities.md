# RP-002 — Future Research Opportunities

These opportunities derive from the five Research Initiatives proposed in DOC-B (SanaLab) and the knowledge frontiers identified in DOC-A (Genspark). They range from Alpha Proxima-executable synthesis work to next-generation neuroscience requiring specialized laboratory infrastructure.

---

## Category A — Alpha Proxima Executable (Synthesis / Design)

### FRO-001 — LUMIAION Memory Architecture Specification
**Source:** DOC-B Initiative D  
**Description:** Develop a complete specification for LUMIAION's three-tier memory system (parametric / transient / external), informed by the human memory architecture. Address: how external memory should be structured, indexed, and maintained; how transient memory can be extended across sessions; what the parametric layer can and cannot store.  
**Output:** Architecture Decision Record (ADR) + System Design document  
**Dependencies:** RP-002 Canonical Synthesis; The Orchestration Framework

### FRO-002 — Collective Memory and Institutional Design
**Source:** DOC-B Initiative E  
**Description:** Apply the anthropological and systems theory frameworks for collective memory to Alpha Proxima's institutional design. How should the vault be structured so that it functions as reliable collective memory rather than a collection of individual session records?  
**Output:** Constitutional provision or institutional design document  
**Dependencies:** RP-002 Comparative Framework; anthropology domain analysis

### FRO-003 — Educational Memory Engineering for LUMIAION
**Source:** DOC-B Initiative C  
**Description:** Apply spacing, retrieval practice, and interleaving principles to LUMIAION's knowledge acquisition design. How should research programs be structured so that findings are consolidated into parametric knowledge, not just filed in the vault?  
**Output:** Research program methodology document  
**Dependencies:** Education domain analysis; RP-002 Theory Matrix

### FRO-004 — Multi-Scale Memory Architecture
**Source:** DOC-B Initiative A  
**Description:** Develop a theoretical framework that maps memory architecture across scales: synaptic → cognitive → cultural → AI. Where are the structural analogies? Where do the analogies break down? What are the design implications of the breakdown points?  
**Output:** Theoretical paper or canonical document  
**Dependencies:** Full RP-002 synthesis; RP-001 AI landscape analysis

## Category B — Requires External Collaboration

### FRO-005 — Contemplative Memory Transformation Study
**Source:** DOC-B Initiative B  
**Description:** Design a study examining whether mindfulness-based interventions operating on autobiographical emotional memories produce effects consistent with reconsolidation (protein synthesis dependency, NMDA receptor involvement, time-limited update window) or with extinction (parallel fear memory trace, context-dependent, prone to renewal).  
**Required:** Neuroscience collaborators; clinical IRB; fMRI + pharmacological tools  
**Timeline:** 2-5 years to publishable results

### FRO-006 — Institutional Memory Decay Study
**Source:** Alpha Proxima operational context  
**Description:** Systematically audit the ALPHA.PROXIMA.CORE- vault at 6-month intervals to measure the rate at which canonical documents drift from accuracy without explicit review. Model the "forgetting curve" of institutional memory.  
**Output:** Empirical governance data; revised review cadence policy  
**Timeline:** Ongoing; 18-month longitudinal study

## Category C — Requires Next-Generation Scientific Tools

### FRO-007 — Engram Code Resolution
**Source:** DOC-A Knowledge Frontier 1  
**Description:** Identify the physical code by which specific memories are represented in engram cell assemblies, using simultaneous large-scale recording (Neuropixels), two-photon calcium imaging, and optogenetic read/write tools.  
**Required:** Neuropixels arrays; two-photon imaging; DREADDs; animal models  
**Timeline:** 10-20 years to fundamental resolution

### FRO-008 — Non-Synaptic Storage Mechanism
**Source:** DOC-A Knowledge Frontier 3  
**Description:** Determine whether epigenetic mechanisms (DNA methylation, histone modifications, ncRNA, PNN scaffolding) are sufficient to explain memory persistence over decades despite synaptic protein turnover in days.  
**Required:** Molecular biology laboratory; single-neuron epigenome sequencing; longitudinal behavioral tracking  
**Timeline:** 5-15 years

### FRO-009 — Glial Memory Contribution Quantification
**Source:** DOC-A Knowledge Frontier 4  
**Description:** Quantify the relative contribution of astrocyte glutamate cycling and microglial synaptic pruning to the specificity of systemic consolidation — i.e., which memories survive sleep consolidation and which are pruned.  
**Required:** Glial-specific transgenic models; optogenetic glia manipulation; single-cell transcriptomics  
**Timeline:** 5-10 years
