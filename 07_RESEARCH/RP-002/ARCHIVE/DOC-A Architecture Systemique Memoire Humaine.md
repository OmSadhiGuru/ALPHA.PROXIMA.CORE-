## Scope

DOC-A is a technical neuroscience reference architecture covering:

1. **Macro-Architecture** — MTL complex (hippocampal formation, perirhinal/parahippocampal cortex, entorhinal cortex), PFC suite (dlPFC, vlPFC, mPFC), subcortical reservoirs (BLA amygdala, striatum, parietal/sensory cortex)
2. **Engram Lifecycle** — Encoding → Consolidation → Storage → Retrieval → Reconsolidation (with ASCII circuit diagrams)
3. **Trisynaptic Circuit** — CE → DG → CA3 → CA1 → Subiculum; pattern separation (DG) and pattern completion (CA3)
4. **Two-Level Consolidation** — Synaptic (LTP/LTD, minutes-hours) + Systemic (neocortical transfer, weeks-years)
5. **Neuroplasticity** — Hebbian (LTP/LTD), synaptic homeostasis (AMPA scaling), structural plasticity, adult neurogenesis (DG subgranular zone)
6. **Oscillatory Dynamics** — Online: theta-gamma coupling (4-8 Hz / 30-100 Hz); Offline: SWR synchronization (150-250 Hz + delta 1-4 Hz + sleep spindles 11-16 Hz)
7. **Cognitive Controls** — Working memory, attentional filtering (dlPFC + FPCN), emotional modulation (HPA → BLA → noradrenergic), predictive processing (mPFC schemas + VTA dopaminergic errors)
8. **Knowledge Frontiers** — Engram code nature, schema integration (CLS theory), non-synaptic/epigenetic storage (PNN, DNA methylation, histone modifications, ncRNA), glial coupling (tripartite synapse: astrocytes + microglia)
9. **Institutional Research Priority Matrix** — 4 frontiers × obstacles × next-generation tools (Neuropixels, fUS, DREADDs, two-photon imaging)
