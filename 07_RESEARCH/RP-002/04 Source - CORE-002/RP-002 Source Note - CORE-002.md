## Actual Contribution

DOC-A delivered a 7-section technical reference architecture covering:

1. **Macro-architecture** — Full topographic map: MTL complex (hippocampal formation, perirhinal cortex, parahippocampal cortex, entorhinal cortex), PFC suite (dlPFC / vlPFC / mPFC), subcortical reservoirs (BLA amygdala, striatum, parietal/sensory cortex)
2. **Trisynaptic circuit specification** — CE → DG → CA3 → CA1 → Subiculum; pattern separation (DG) vs pattern completion (CA3) — computationally distinct operations
3. **Two-level consolidation model** — Synaptic: LTP, NMDA/AMPA receptor dynamics, CaMKII, mTOR pathway, dendritic spine remodeling (minutes-hours); Systemic: hippocampal index → neocortical transfer (weeks-years)
4. **Reconsolidation mechanism** — NMDA activation → ubiquitin-proteasome degradation → ZIP/PKMζ restabilization; defines the therapeutic modification window
5. **Oscillatory dynamics** — Online encoding: theta-gamma coupling (4-8 Hz / 30-100 Hz, phase-amplitude); Offline consolidation: SWR synchronization (150-250 Hz) entrained with delta (1-4 Hz) + thalamic sleep spindles (11-16 Hz)
6. **Four neuroplasticity types** — Hebbian LTP/LTD, synaptic homeostasis (AMPA scaling), structural plasticity (actin cytoskeleton, dendritic spine morphogenesis), adult neurogenesis (DG subgranular zone)
7. **Knowledge frontier taxonomy** — Four open frontiers: engram code nature, schema integration (CLS theory), non-synaptic/epigenetic storage (PNN scaffolding, DNA methylation, histone modifications, ncRNA), glial coupling (tripartite synapse with astrocytes + microglia active in synaptic pruning during SWS)
8. **Research priority matrix** — Each frontier matched to experimental obstacles and next-generation tools

## Key Conflicts with Other Sources

None identified at the factual level. DOC-A and DOC-B are complementary in scope: DOC-A goes deeper on neuroscience mechanisms; DOC-B goes broader across disciplines. Apparent tensions:

- DOC-A presents Standard Consolidation Theory (Squire) and Multiple Trace Theory as both active models — correctly reflecting genuine empirical competition
- DOC-A raises the non-synaptic storage paradox (synaptic protein turnover in days vs memory persistence over decades) as an unresolved frontier — this is a genuine open question that DOC-B does not cover at this depth
