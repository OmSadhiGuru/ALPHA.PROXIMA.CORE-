# Concept Note — Hippocampus

---

## Canonical Definition

The **hippocampus** is a bilateral medial temporal lobe structure required for the formation of new declarative memories [C-003]. It consists of the hippocampal formation proper (CA1, CA3, dentate gyrus, subiculum) plus its primary input/output interface (entorhinal cortex). The hippocampus does not store memories permanently — it acts as an indexing structure that binds distributed neocortical representations during encoding and coordinates their transfer during systemic consolidation.

---

## Subfield Architecture

```
Entorhinal Cortex (EC) — input/output gateway
        ↓ Perforant Path
Dentate Gyrus (DG) — pattern separation; adult neurogenesis
        ↓ Mossy Fibers
CA3 — pattern completion; associative recall via recurrent collaterals
        ↓ Schaffer Collaterals
CA1 — integration and mismatch detection
        ↓
Subiculum — primary output; projects back to EC
```

| Function | Subfield | Mechanism |
|---------|---------|-----------|
| Pattern separation | DG | Sparse coding; orthogonalization of similar inputs |
| Pattern completion | CA3 | Autoassociative recurrent network |
| Mismatch detection | CA1 | Compares EC input with CA3 output |
| Systemic consolidation | CA3 → CA1 | SWR generation during sleep; replay to neocortex |
| Adult neurogenesis | DG (SGZ) | New granule cells increase pattern separation capacity |

## Contested Territory [M-001]

Does old hippocampal memory remain hippocampus-dependent?

- **Standard Consolidation Theory (SCT):** Once systemically consolidated, episodic and semantic memories become hippocampus-independent
- **Multiple Trace Theory (MTT):** Episodic memories always require hippocampus; only decontextualized semantic memories become hippocampus-independent

Both models have empirical support. The HM case study (bilateral hippocampal resection → anterograde amnesia) established hippocampal necessity for *new* declarative memory formation; the retrograde amnesia pattern is less clear and is the core of the SCT/MTT debate.
