# Concept Note — Collective Memory

---

## Canonical Definition

**Collective memory** is the shared stock of representations about the past held by a community, constituted through social frameworks, practices, and institutions. First systematically theorized by Maurice Halbwachs (1925). Collective memory is **irreducible to the sum of individual memories** — it is socially constituted through interaction, not merely aggregated from individual recollections.

---

## Key Properties

1. **Socially constituted** — requires social interaction, shared frameworks, and institutional maintenance to exist
2. **Selective** — communities remember selectively, emphasizing what supports current identity and purpose
3. **Mutable** — collective memories are revised, reinterpreted, and contested; especially across generational transitions
4. **Distributed** — stored in multiple media: oral tradition, writing, monuments, rituals, practices, institutions
5. **Identity-constituting** — collective memory shapes what a group understands itself to be

---

## Memory Media (Anthropology / History)

| Medium | Function | Fragility |
|--------|---------|-----------|
| Oral tradition | Distributed; self-maintaining through performance | Disappears when transmission chain breaks |
| Written documents | Persistent; accurate for text; decontextualized | Requires literacy and reading practice |
| Material artifacts / monuments | Durable; spatially anchored; low information density | Physical destruction; meaning can be lost |
| Institutional practices | Embedded in behavior; high fidelity to function | Institution must survive; practices can drift |
| Digital records | Massive; searchable; fragile without maintenance | Bit rot; format obsolescence; link rot |

## Alpha Proxima Application

Alpha Proxima's collective memory is constituted by:

| Memory Component | Medium | Maintenance Requirement |
|-----------------|--------|------------------------|
| Canonical vault (ALPHA.PROXIMA.CORE-) | Digital documents (git) | Regular review, versioning, maintenance |
| LUMIAION parametric memory | Model weights | Cannot be directly inspected or updated without retraining |
| Founding member biographical memory | Personal (irreplaceable) | Capture in vault before natural loss |
| Governance documents | Formal written documents | Explicit versioning; amendment procedures |
| Institutional practices | Embedded in workflows | Must be documented when established |

**Critical vulnerability:** The founding member's biographical memory (the irreplaceable context behind Alpha Proxima's origins) exists only in personal memory. If not captured in the vault, it cannot be reconstructed. This is the institutional equivalent of collective memory dependent on a single oral tradition holder.
