# Concept Note — Reconsolidation

---

## Canonical Definition

**Reconsolidation** is the process by which a retrieved memory re-enters a labile, chemically unstable state requiring active restabilization through new protein synthesis to persist. Described formally by Nader, Schafe, and LeDoux (2000) in rodent fear conditioning. The reconsolidation window is approximately 6 hours post-retrieval [C-008].

Reconsolidation is the molecular mechanism underlying the cross-disciplinary consensus that memory is reconstructive [C-001] — the past is modifiable every time it is accessed.

---

## Molecular Mechanism

```
1. Memory cue → successful retrieval
        ↓
2. NMDA receptor reactivation in the MTL
        ↓
3. Ubiquitin-proteasome pathway activated
   → degradation of stabilizing proteins (including PKMζ substrate)
   → memory enters labile state
        ↓
4. ~6-hour window of vulnerability
   (modification, strengthening, or erasure possible)
        ↓
5. New protein synthesis (ZIP/PKMζ reactivation)
   → memory restabilized in updated form
        ↓
6. Memory reconsolidated — now reflects retrieval context
```

---

## Consequences

| Consequence | Implication |
|-------------|-------------|
| Memory is modified during retrieval | Every recall is also a potential update |
| Reconsolidation window is time-limited (~6 hours) | Therapeutic or disruptive interventions must occur within window |
| Retrieval without restabilization → degradation | Inhibiting protein synthesis after retrieval weakens memory |
| Memories strengthen or update depending on retrieval context | Context of recall shapes what is consolidated |

---

## Therapeutic Implications [Q-004]

The reconsolidation window is a target for:
- **PTSD treatment:** Reactivating traumatic memory in safe context → allowing reconsolidation with reduced emotional charge
- **Fear extinction vs. reconsolidation:** Extinction creates a new inhibitory trace; reconsolidation directly modifies the original trace. Different mechanisms; different relapse profiles.
- **Propranolol (beta blocker):** Administered during reconsolidation window reduces physiological fear response to reactivated trauma memories

Clinical protocols are not yet standardized. Reliable targeting of the reconsolidation window (vs. extinction) in humans remains an open problem [Q-004].
