# Concept Note — Memory

---

## Canonical Definition

**Memory** is the capacity to acquire, retain, and reconstruct information across time in ways that modify future behavior, perception, and identity. It operates simultaneously at multiple scales: molecular (synaptic weight changes), neural circuit (oscillatory dynamics), cognitive (schemas, narrative), and cultural (institutions, transmitted knowledge).

The defining feature of memory is that it is **reconstructive, not reproductive** [C-001] — every act of recall re-creates a memory rather than playing back a stored recording. This is not a limitation; it is the architecture that enables memory to be updated, integrated, and contextualized.

---

## Key Properties

1. **Reconstructive** — retrieval re-creates; does not play back [C-001]
2. **Context-dependent** — encoding context shapes retrieval [C-010]
3. **Modifiable** — reconsolidation window opens during retrieval [C-008]
4. **Consolidation-requiring** — two-phase stabilization (synaptic + systemic) [C-004]
5. **Sleep-dependent** — offline processing is mandatory [C-005]
6. **Emotionally gated** — BLA amplifies encoding of salient events [C-006]
7. **Socially constituted** — collective memory is irreducible to individual memories [DOC-B]
