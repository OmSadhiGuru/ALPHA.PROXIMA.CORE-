## Program Scope (Planned)

RP-002 will map the scientific and philosophical understanding of human memory — its neural substrates, computational mechanisms, and implications for AI system design.

**Anticipated coverage:**
- Memory systems (episodic, semantic, procedural, working)
- Memory consolidation (hippocampus, sleep, synaptic consolidation)
- Retrieval mechanisms (pattern completion, reconstruction)
- Memory and identity
- Computational models of memory (Hopfield networks, complementary learning systems, transformers)
- Implications for AI long-term memory architectures

---

## Relationship to RP-001

RP-002 is the natural successor to RP-001. Consciousness and memory are deeply intertwined:
- Working memory is a key component of the GNWT global workspace
- Episodic memory requires consciousness (tulving's episodic memory hypothesis)
- Autobiographical memory is central to the Default Mode Network and the DMN's role in self
