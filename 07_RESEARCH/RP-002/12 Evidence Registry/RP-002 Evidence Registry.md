# RP-002 — Evidence Registry

---

## Classification System

| Code | Class | Definition |
|------|-------|------------|
| **C** | Consensus | Replicated, cross-disciplinary empirical consensus; no serious scientific challenge |
| **M** | Competing Models | Multiple credible frameworks; genuine empirical competition unresolved |
| **E** | Emerging Evidence | Recent findings; promising but not yet replicated at consensus level |
| **Q** | Open Question | Known unknown; question is well-defined but answer is genuinely unresolved |
| **S** | Speculative | Forward-looking conjecture; limited current empirical constraint |
| **P** | Phenomenological | First-person / experiential evidence; not reducible to third-person measurement |

## Consensus (C)

| ID | Claim | Source | Notes |
|----|-------|--------|-------|
| C-001 | Memory is reconstructive, not reproductive — every retrieval re-creates rather than replays | DOC-B (cross-disciplinary); DOC-A (reconsolidation mechanism) | Bartlett 1932; confirmed at molecular level by reconsolidation research |
| C-002 | Long-term potentiation (LTP) is the primary cellular mechanism underlying synaptic memory formation | DOC-A | Bliss & Lømo 1973; NMDA/AMPA receptor dynamics; replicated across species |
| C-003 | The hippocampus is required for formation of new declarative (explicit) memories | DOC-A, DOC-B | HM case study; MTL lesion literature; consensus across neuroscience and cognitive psychology |
| C-004 | Memory consolidation requires two phases: synaptic (minutes-hours) and systemic (weeks-years, neocortical transfer) | DOC-A | Standard Consolidation Theory; mechanistic support from SWR, sleep spindle, delta oscillation research |
| C-005 | Sleep is mandatory for memory consolidation — offline processing is not optional | DOC-A, DOC-B | SWR synchronization during NREM; theta oscillations during REM; cross-disciplinary consensus |
| C-006 | Emotional salience modulates memory encoding strength via amygdala (BLA) — noradrenergic and glucocorticoid pathways | DOC-A | HPA → BLA → noradrenergic cascade; flashbulb memory at behavioral level (DOC-B) |

| C-010 | Encoding context shapes retrieval — memory is context-dependent across all studied species and disciplines | DOC-B | Cross-disciplinary consensus; encoding specificity principle |

## Competing Models (M)

| ID | Claim | Models in Competition | Source | Notes |
|----|-------|----------------------|--------|-------|
| M-001 | How does long-term memory depend on the hippocampus over time? | Standard Consolidation Theory (Squire — hippocampus required only during consolidation, then independent) vs. Multiple Trace Theory (Nadel/Moscovitch — hippocampus always required for episodic memory, regardless of age) | DOC-A | Genuine empirical competition; both models have supporting evidence; unresolved |
| M-002 | Does the hippocampus primarily compute rapid pattern separation or flexible relational binding? | Pattern separation emphasis (DG subfield focus) vs. Relational/cognitive map theory (O'Keefe/Moser; place cells, grid cells) | DOC-A | Two non-exclusive but differently weighted mechanistic accounts |
| M-003 | Is synaptic plasticity the only storage medium, or do non-synaptic mechanisms contribute significantly? | Synaptic-only model vs. Non-synaptic/epigenetic model (DNA methylation, histone modifications, ncRNA, PNN scaffolding) | DOC-A | Synaptic protein turnover paradox (days vs. decades) creates pressure on synaptic-only model; emerging evidence for epigenetic mechanisms |
| M-004 | What is the correct theory of schema integration? | Complementary Learning Systems (fast hippocampal acquisition + slow neocortical integration) vs. Direct schema-guided encoding models (mPFC schemas constrain hippocampal encoding without two-stage transfer) | DOC-A | CLS theory widely cited; direct encoding view is emerging; DOC-A marks this as a knowledge frontier |

## Emerging Evidence (E)

| ID | Claim | Source | Evidence Status |
|----|-------|--------|-----------------|
| E-001 | Glial cells (astrocytes + microglia) are active participants in synaptic plasticity, not passive support | DOC-A | Tripartite synapse model: astrocytes (glutamate clearance, D-serine co-agonist, glycogen-lactate shuttle) and microglia (synaptic pruning during SWS) — mechanism identified; functional significance at memory systems level still emerging |
| E-002 | Epigenetic mechanisms (DNA methylation, histone modifications, ncRNA) may encode memory in ways that persist beyond synaptic protein turnover | DOC-A | Perineuronal net (PNN) scaffolding implicated; experimental support exists but mechanism not yet integrated into mainstream consolidation models |
| E-003 | Adult neurogenesis in the dentate gyrus subgranular zone contributes to pattern separation capacity and cognitive flexibility | DOC-A, DOC-B | Evidence from rodent studies; human adult neurogenesis remains contested; functional role emerging |
| E-004 | Contemplative practices (mindfulness, meditation) produce measurable changes in memory encoding and emotional memory processing beyond relaxation effects | DOC-B | Research initiative B in DOC-B; mechanistic intersection with reconsolidation window noted; evidence accumulating but not consolidated |
| E-005 | Theta-gamma coupling (phase-amplitude coupling) in the hippocampus implements a temporal code for working memory items during online encoding | DOC-A | 4-8 Hz theta / 30-100 Hz gamma; electrophysiological evidence strong in rodents and primates; functional significance in humans emerging |

## Open Questions (Q)

| ID | Question | Source | Notes |
|----|---------|--------|-------|
| Q-001 | What is the nature of the engram code? How is a specific memory physically instantiated? | DOC-A | Most fundamental open question in memory neuroscience; optogenetic tagging has identified engram cells but the code (pattern of weights? topology? oscillatory signature?) remains unknown |
| Q-002 | Does adult hippocampal neurogenesis occur at functionally significant rates in adult humans? | DOC-A, DOC-B | Rodent evidence strong; human evidence contested — some postmortem studies find very low rates; technology limitations complicate in vivo measurement |
| Q-003 | How does schema integration work mechanistically? | DOC-A | mPFC schemas participate in encoding but the precise circuit mechanism by which prior knowledge gates new encoding remains uncharacterized at cellular resolution |
| Q-004 | Can the reconsolidation window be reliably and ethically directed for therapeutic memory modification in clinical settings? | DOC-B | Reconsolidation is real; therapeutic targeting (PTSD, phobia, addiction) shows promise but protocols are not yet standardized or approved at scale |
| Q-005 | What is the relationship between individual neural memory and collective cultural memory — is there a common underlying architecture? | DOC-B | Anthropology / systems theory framing; currently a theoretical bridge rather than an empirical question; may require new measurement tools |
