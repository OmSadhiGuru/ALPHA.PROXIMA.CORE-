# Workflows

## Research Note Workflow

Input: research question, source list, scope.

Steps:

1. Define the question.
2. Gather sources.
3. Evaluate source quality.
4. Summarize findings.
5. State uncertainty.
6. Route durable findings to `KNOWLEDGE/`.

Output: research note or source evaluation.
