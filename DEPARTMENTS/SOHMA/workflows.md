# Workflows

## Report Workflow

Input: reflective source material and reporting period.

Steps:

1. Identify source boundaries.
2. Extract recurring themes.
3. Mark uncertainty.
4. Produce structured report.
5. Route durable insights to knowledge capture.

Output: evolution report or analysis note.
