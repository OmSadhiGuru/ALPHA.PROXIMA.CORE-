# OSG Content Area

## Purpose

Organize public content from idea to publication.

## Recommended Hierarchy

```text
05_CONTENT/
  README.md
  ideas/
  drafts/
  review/
  scheduled/
  published/
  repurposed/
  archive/
```

## Rules

- Every content item should have one status.
- Every published item should link to its source draft and public URL.
- Content should have a CTA connected to a course, booking, email list, or offer.
