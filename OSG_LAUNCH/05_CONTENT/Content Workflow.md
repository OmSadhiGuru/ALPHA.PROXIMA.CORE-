# OSG Content Workflow

## Purpose

Define the minimum content pipeline required for launch.

## Workflow

```text
Idea
  -> Draft
  -> Edit
  -> Asset
  -> Schedule
  -> Publish
  -> Repurpose
  -> Measure
```

## Required Fields

- title
- channel
- status
- target audience
- CTA
- publish date
- source asset
- public URL
- result metric

## Launch Channels

Start with only the channels that can be maintained consistently.

Recommended launch baseline:

- one primary social channel
- email list
- website or landing page
- course/client conversion page

- [ ] Add AI-assisted draft generation after voice and offer are stable.
