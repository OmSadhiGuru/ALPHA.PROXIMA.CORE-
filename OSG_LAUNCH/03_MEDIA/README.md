# OSG Media Area

## Purpose

Organize launch media assets for courses, content, website, and client delivery.

## Recommended Hierarchy

```text

  README.md
  brand/
  video/
  audio/
  images/
  thumbnails/
  exports/
```

## Rules

- Keep source files separate from exports.
- Include usage rights and source notes for externally sourced assets.
- Use date and channel in filenames for public content exports.
