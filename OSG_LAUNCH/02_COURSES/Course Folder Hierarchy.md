# OSG Course Folder Hierarchy

## Purpose

Define the reusable folder pattern for every OSG course.

## Course Folder

```text
COURSE-SLUG/
  README.md
  00_overview.md
  01_curriculum.md
  02_modules/
  03_lessons/
  04_scripts/
  05_worksheets/
  06_slides/
  07_media/
  08_sales_assets/
  09_delivery/
  99_archive/
```

## Required Course README Sections

- offer
- audience
- outcome
- modules
- production status
- launch status
- asset links
- known blockers
