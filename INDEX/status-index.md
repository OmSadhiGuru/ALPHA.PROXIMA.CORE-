# Status Index

| Status | Meaning | Action |
| --- | --- | --- |
| draft | Work in progress | Do not treat as final |
| active | Current working standard | Use normally |
| review | Awaiting review | Verify before use |
| approved | Approved decision or standard | Use as authoritative |
| superseded | Replaced by newer document | Follow related replacement |
| archived | Retired institutional memory | Preserve for history |
