# LOGS

## Purpose

Store operational logs, handoff notes, sync logs, approval records, and incident notes.

## Scope

Logs preserve operational history. Durable insights should be promoted from logs into `KNOWLEDGE/`.

## Inputs

- Handoffs.
- Sync runs.
- Approval decisions.
- Incident notes.

## Outputs

- Audit trail.
- Operational history.
- Knowledge promotion candidates.
